# Fiche 01 : Structure cellulaire et domaines du vivant

**Date de creation** : 14 novembre 2025  
**Sujet** : Cellules, noyau, chromosomes, eucaryotes vs procaryotes

---

## Hierarchie d'organisation

```
Cellule → Noyau → Chromatine → Chromosomes → ADN/ARN → Nucleotides
```

---

## Definitions cles

### Cellule
- Unite de base de tout materiel vivant
- "Building block of life"

### Noyau (eucaryotes)
- Composant central identifie il y a 150 ans
- Hub de l'activite cellulaire
- Apparait comme une region plus sombre au microscope

### Chromatine
- Melange dense de macromolecules dans le noyau
- Contient les acides nucleiques

### Chromosomes
- Forme condensee de la chromatine pendant la mitose
- Longues chaines minces
- Contiennent l'ensemble du genome

---

## Les trois domaines du vivant

### 1. Eucaryotes (Eukaryota)

**Caracteristique definissante** : cellules contenant des organelles complexes entourees de membranes, notamment le **noyau cellulaire**

**Proprietes** :
- Comprennent tous les organismes multicellulaires + certains unicellulaires
- Ces organelles permettent une division du travail dans la cellule
- Cette organisation facilite des taches cellulaires complexes
- A permis l'evolution des organismes unicellulaires vers des organismes multicellulaires

**Exemples** :
- Animaux (humains, insectes, poissons)
- Plantes (arbres, fleurs, algues vertes)
- Champignons (levures, moisissures)
- Protistes (amibes, paramecies)

### 2. Bacteria (Bacteries)

**Caracteristique definissante** : organismes procaryotes sans organelles membranaires

**Proprietes** :
- Tous unicellulaires
- Pas de noyau : l'ADN n'est pas enferme dans une membrane
- ADN typiquement forme d'un seul chromosome circulaire
- Division cellulaire simple : **fission binaire**

**Exemples** :
- E. coli (intestin)
- Streptococcus (infections)
- Cyanobacteries (photosynthese)

### 3. Archaea (Archees)

**Caracteristique definissante** : organismes procaryotes similaires aux bacteries mais avec des differences moleculaires importantes

**Proprietes** :
- Tous unicellulaires
- Pas de noyau ni d'organelles
- Souvent extremophiles (environnements extremes)
- Biochimie distincte des bacteries

**Exemples** :
- Organismes thermophiles (sources chaudes)
- Halophiles (lacs sales)
- Methanogenes (production de methane)

---

## Comparaison Eucaryotes vs Procaryotes

| Caracteristique | Eucaryotes | Procaryotes (Bacteria + Archaea) |
|-----------------|------------|----------------------------------|
| **Domaines** | Eukaryota | Bacteria + Archaea |
| **Complexite** | Unicellulaires + **Tous les multicellulaires** | Unicellulaires uniquement |
| **Noyau** | Present (ADN enferme) | Absent (ADN dans le cytoplasme) |
| **Organelles** | Presentes (mitochondries, chloroplastes, etc.) | Absentes |
| **ADN** | Lineaire, plusieurs chromosomes | Circulaire, typiquement 1 chromosome |
| **Division** | Mitose (complexe) | Fission binaire (simple) |
| **Taille** | Generalement plus grandes (10-100 µm) | Plus petites (1-10 µm) |
| **Exemples** | Humains, arbres, champignons | E. coli, archees extremophiles |

## Points cles a retenir

### Structure hierarchique
- Cellule → Noyau → Chromatine → Chromosomes → Acides nucleiques

### Trois domaines
- **Eukaryota** : noyau + organelles (tous les multicellulaires)
- **Bacteria** : procaryotes, pas de noyau
- **Archaea** : procaryotes extremophiles

### Difference fondamentale
- **Eucaryotes** : compartimentalisation (organelles)
- **Procaryotes** : simplicite structurale

### Evolution
- Procaryotes → premiers organismes (3.5 milliards d'annees)
- Eucaryotes → evolution plus recente (2 milliards d'annees)
- Multicellularite → uniquement chez les eucaryotes

## Mnemoniques

**Trois domaines** : "**E**milie **B**oit **A**vec" (Eukaryota, Bacteria, Archaea)

**Eucaryotes** : "**E**nveloppe le **N**oyau" (Eucaryotes = Noyau)

**Procaryotes** : "**P**as **O**rganise" (Pas d'Organelles)

**Difference cle** : 
- **E**ucaryotes = **E**nferme (noyau enferme l'ADN)
- **P**rocaryotes = **P**as enferme (ADN libre)

## Lien avec la bio-informatique

**Sequencage de genomes** :
- Genomes procaryotes : plus petits, plus simples (3-5 millions de paires de bases)
- Genomes eucaryotes : plus grands, plus complexes (3 milliards pour humain)

**Annotation de genomes** :
- Procaryotes : genes densement packes, peu d'introns
- Eucaryotes : beaucoup d'introns, epissage alternatif

**Metagenomique** :
- Etude de communautes microbiennes (bacteries + archees)
- Identification d'especes par sequencage

**Comparaison evolutive** :
- Arbres phylogenetiques basés sur sequences d'ARN ribosomal
- Reconstruction de l'histoire evolutive
