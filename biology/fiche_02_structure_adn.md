## Les acides nucleiques

### Caracteristiques generales

**Definition**
- Polymeres : chaines repetitives de molecules similaires (monomeres)
- Forme : longues et minces → appeles "brins" (strands)

**Unite de mesure**
- **Nucleotide (nt)** = unite de longueur des brins

### Structure d'un nucleotide

Un nucleotide est forme de **3 parties** :

1. **Sucre** (sugar)
   - ADN : desoxyribose
   - ARN : ribose

2. **Phosphate**
   - Ion charge negativement

3. **Base azotee** (nucleobase)
   - C'est la partie variable !

### Polymerisation

**Liaison**
- Sucre d'un nucleotide ↔ Phosphate du nucleotide suivant
- Forme un **squelette sucre-phosphate** (sugar-phosphate backbone)

**Structure primaire**
- Definie uniquement par l'**ordre des bases**
- Sucre et phosphate identiques dans un meme type d'acide nucleique
- Seules les bases different

---

## ADN (Acide DesoxyriboNucleique)

### Composition

**Sucre** : Desoxyribose

**4 bases azotees** :
- **A** : Adenine
- **C** : Cytosine
- **G** : Guanine
- **T** : Thymine

### Localisation et role

**Present dans** :
- Tous les organismes vivants (bacteries incluses)
- De nombreux virus (meme consideres non-vivants)

**Genome**
- Terme reserve a l'ADN total d'un organisme
- Somme de tout l'ADN contenu dans les chromosomes

**Dans les eucaryotes** :
- Reste confine dans le **noyau**
- Sert de **modele** pour creer l'ARN

---

## Structure primaire de l'ADN

### Definition
- **Ordre des bases** le long du squelette sucre-phosphate
- Dit QUOI mais pas COMMENT c'est organise en 3D

### Representation
```
5' - A - T - C - G - A - T - C - G - 3'
     |   |   |   |   |   |   |   |
  sucre-P-sucre-P-sucre-P-sucre-P-sucre
```

### Notation
- Direction 5' → 3' (sens conventionnel)
- 5' = extremite avec groupe phosphate libre
- 3' = extremite avec groupe hydroxyle libre

### Exemple de sequence
```
5' - ATCGATCGTAGC - 3'
```

## Structure secondaire de l'ADN

### Decouverte historique
- **1953** : Watson & Crick (publication Nature, <800 mots)
- Basee sur image X-ray haute resolution de **Rosalind Franklin** & Raymond Gosling
- Consolidation de resultats chimiques etablis

### 3 principes fondamentaux

#### 1. Double brin antiparallele
- Molecule d'ADN = **2 brins**
- Courent dans des **directions opposees**
- Un brin 5'→3', l'autre 3'→5'

#### 2. Appariement complementaire des bases
- Chaque base se lie a une base du brin oppose
- Regles strictes d'appariement :
  - **A ↔ T** (Adenine avec Thymine) : 2 liaisons hydrogene
  - **C ↔ G** (Cytosine avec Guanine) : 3 liaisons hydrogene
- **Complement** = base a laquelle une base donnee se lie toujours

#### 3. Organisation en double helice
- Voir structure tertiaire

### Consequences de la complementarite

**Paire de bases (bp)**
- Liaison de 2 bases complementaires = 1 paire de bases
- **Nouvelle unite de mesure** : bp (au lieu de nt)
- Longueur ADN souvent exprimee en bp

**Deduction de sequence**
- Si on connait un brin → on peut deduire l'autre
- Les bases du brin complementaire sont dans l'**ordre inverse**

**Exemple** :
```
Brin 1 (5'→3') :  5' - ATCGATCG - 3'
Brin 2 (3'→5') :  3' - TAGCTAGC - 5'

Lecture brin 2 de gauche a droite : CGAT CGAT
```

**Complement reverse (reverse complement)** :
- Sequence du brin 1 : ATCGATCG
- Complement : TAGCTAGC
- Complement reverse (brin 2 lu 5'→3') : CGATCGAT

## Structure tertiaire de l'ADN

### La double helice

**Description**
- Les 2 brins sont **torsades ensemble**
- Forme un long "escalier en spirale" (spiral staircase)
- Structure 3D de la molecule d'ADN

**Caracteristiques** :
- **Marches de l'escalier** : paires de bases (A-T ou C-G)
- **Rampes** : squelettes sucre-phosphate
- **Spirale** : helice droite (forme B la plus courante)

**Avantages de cette structure** :
- Compactage de l'information
- Protection des bases a l'interieur
- Stabilite de la molecule
- Facilite la replication

### Dimensions
- Diametre : ~2 nanometres (nm)
- 1 tour complet : ~3.4 nm
- ~10 paires de bases par tour

## Points cles a retenir

### Nucleotides
- 3 composants : Sucre + Phosphate + Base
- Structure primaire = ordre des bases uniquement

### ADN
- 4 bases : A, C, G, T
- Sucre : desoxyribose
- Reste dans le noyau (eucaryotes)
- Stocke l'information genetique = Genome

### Structure secondaire
- 2 brins antiparalleles
- Appariement complementaire : A-T et C-G
- Unite de mesure : paire de bases (bp)

### Structure tertiaire
- Double helice
- Bases protegees a l'interieur
- Structure stable et compacte

### Complement reverse
- Important en bio-informatique
- Brin 1 : ATCG
- Complement : TAGC
- Complement reverse : CGAT

## Mnemoniques

**Bases ADN** : "**A** **C**T **G**rand **T**rain"
- Adenine, Cytosine, Guanine, Thymine

**Sucre** :
- ADN = **D**esoxy**D**NA
- Le "D" de DNA rappelle Desoxy

**Appariement** :
- **A**-**T** : "**A**lways **T**ogether"
- **C**-**G** : "**C**ompletely **G**lued"
- Ou : A et T ont 2 points de contact, C et G en ont 3

**Structures** :
- **P**rimaire = ordre des **P**erlites (bases)
- **S**econdaire = **S**ysteme a 2 brins
- **T**ertiaire = **T**orsade (helice)

**Direction** :
- **5** est impair → commence **p**ar P (phosphate)
- **3** est impair → finit par OH

## Lien avec la bio-informatique

Ces concepts sont **fondamentaux** pour comprendre :

### Sequencage
- Determiner l'ordre des bases (A, C, G, T)
- Technologies : Illumina, Nanopore, PacBio
- Output : fichiers FASTA, FASTQ

### Complement inverse
- Calculer le brin complementaire d'une sequence
- Crucial pour l'assemblage de genomes
- Important pour le mapping des reads

### Alignement de sequences
- Comparer des brins d'ADN/ARN
- Algorithmes : Needleman-Wunsch, Smith-Waterman
- Trouver les regions similaires

### Assemblage de genomes
- Reconstituer des chromosomes a partir de fragments
- Utilise la complementarite des brins
- Graphes de De Bruijn

### Prediction de structures
- Utiliser les regles d'appariement
- Structures secondaires d'ARN
- Energie de liaison

### Design de primers (PCR)
- Creer des sequences complementaires
- Specificite base sur appariement A-T, C-G
- Temperature de melting (Tm)
